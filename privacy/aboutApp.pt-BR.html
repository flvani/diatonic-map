Sobre esta aplicação

  

Diatonic App Desenvolvido por: Flávio Vani
Coordenação Musical: Cezar Ferreira

Este projeto pretende ser um mapa de referência, tanto para a aprendizagem como para o registo histórico do acordeão diatônico. Com este aplicativo é possível configurar - e apresentar - o layout de qualquer acordeão diatônico, em seus mais diversos modelos. A coordenação musical ficou a cargo do prof. Cezar Ferreira – Sala da Gaita Ponto (Facebook), que forneceu o material necessário para a criação dos layouts e as informações essenciais sobre tablaturas para acordeão, além de músicas e exercícios para praticar diretamente no instrumento.

Para transformar o projeto em uma ferramenta de aprendizagem, adotamos uma biblioteca gratuita que permite editar partituras no formato ABC (abcjs) diretamente no navegador. A biblioteca foi estendida para incluir uma ferramenta de transposição de tons e principalmente expandimos a notação ABC (non standard) para incluir uma sintaxe para a produção de tablaturas para acordeões, fundindo o modelo ABC com os mapas de acordeão apresentados aqui.

Por fim, por meio da implementação de bibliotecas MIDI para navegadores (MIDI.js), tornou-se possível ouvir os sons produzidos pelo acordeão, na forma de músicas e exercícios. Esse recurso funciona até mesmo em dispositivos móveis, como telefones Android. Muitas outras bibliotecas ajudaram este projeto a ter sucesso, conforme mostrado abaixo.

O objetivo final é proporcionar uma experiência visual e interativa para quem está iniciando seus estudos no "Acordeão Diatônico".

Meus mais sinceros agradecimentos aos autores dessas bibliotecas:

  • accordion-map de Leandro Nunes: "Um mapa de referência interativo para acordes de acordeão diatônicos".
  • abcjs de Paul Rosen: "Javascript para renderização de notação musical ABC".
  • MIDI.js de Michael Deal: "Facilitando a criação de um aplicativo MIDI na web".
  • ACE Editor por ajax.org: "Um editor de código embutido escrito em JavaScript".
  • HTML5-Color-Picker de Rubchuk Vladimir: "Um selecionador de cores escalonável que se anexa aos elementos DOM".
  • Water Bug de Bernhard Zwischenbrugger: "Um console Javascript simples para dispositivos móveis".
  • Perfect Scrollbar de Hyunje Jun: "Plugin de barra de rolagem personalizado minimalista, mas perfeito".
  • EnjoyHint de XB Software: "Uma maneira simples de criar tutoriais interativos e dicas para seu site ou aplicativo web".
