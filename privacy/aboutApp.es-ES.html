Acerca de esta aplicación

  

Diatonic App Desarrollado por: Flávio Vani
Coordinación Musical: Cezar Ferreira

Este proyecto pretende ser un mapa de referencia, tanto para el aprendizaje como para el registro histórico del acordeón diatónico. Con esta aplicación, es posible configurar - y presentar - el diseño de cualquier acordeón diatónico, en sus diversos modelos. La coordinación musical estuvo a cargo del prof. Cezar Ferreira – Sala da Gaita Ponto (Facebook), quien proporcionó el material necesario para la creación de los diseños y la información esencial sobre tablaturas para acordeón, además de música y ejercicios para practicar directamente en el instrumento.

Para convertir el proyecto en una herramienta de aprendizaje, adoptamos una biblioteca gratuita que permite editar partituras en formato ABC (abcjs) directamente en el navegador. La biblioteca se extendió para incluir una herramienta de transposición de tonos y, principalmente, expandimos la notación ABC (no estándar) para incluir una sintaxis para la producción de tablaturas para acordeones, fusionando el modelo ABC con los mapas de acordeón presentados aquí.

Finalmente, a través de la implementación de bibliotecas MIDI para navegadores (MIDI.js), se hizo posible escuchar los sonidos producidos por el acordeón, en forma de música y ejercicios. Este recurso funciona incluso en dispositivos móviles, como teléfonos Android. Muchas otras bibliotecas ayudaron a que este proyecto tuviera éxito, como se muestra a continuación.

El objetivo final es proporcionar una experiencia visual e interactiva para quienes están comenzando sus estudios en el "Acordeón Diatónico".
