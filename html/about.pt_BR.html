Diatonic-Map 5.00 Desenvolvido por: Flávio Vani
Coordenação musical: Cezar Ferreira

Este projeto é um mapa de referência, tanto para aprendizagem quanto para registro, sobre acordeões diatônicos. Com ele é possível configurar – e mostrar – o layout de qualquer gaita de botões, nos seus mais diversos modelos. A coordenação musical ficou à cargo do prof. Cezar Ferreira - Sala da Gaita Ponto (Facebook), que forneceu o material necessário para criação dos layouts e as essenciais informações sobre tablaturas para acordeões, bem como, músicas e exercícios para praticar diretamente na gaita.

Para transformar o projeto em uma ferramenta de aprendizagem, juntamos uma biblioteca que permite editar partituras no formato ABC (abcjs) diretamente no browser. O editor foi estendido para incluir uma ferramenta de transposição de tonalidades e, principalmente, ampliamos a notação ABC (non-standard) para incluir uma sintaxe para produção de tablaturas para acordeões, juntando o modelo ABC com os mapas de acordeões aqui apresentados.

Por fim, através da implementação das bibliotecas de MIDI para browsers (MIDI.js), tornou-se possível ouvir os sons produzidos pelo acordeão, na forma de canções e exercícios. Este recurso funciona inclusive em dispositivos móveis, tais como, celulares Android.

O objetivo final é proporcionar uma experiência visual e interativa para aqueles que estão iniciando seus estudos na "Gaita Ponto".
